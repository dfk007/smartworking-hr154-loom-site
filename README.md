# SmartWorking DevOps GCP HR154

Workspace for preparing the Smart Working Loom submission for `DevOps Engineer - Google Cloud Platform & Terraform [HR154]`.

## Main Deliverable

The primary file intended for review and deployment is:

- `2-opencode-loom/opencode-loom-script-final.html`

That HTML acts as the master hub and includes:

- sample-video analysis
- OpenCode's tailored script and guidance
- rehearsal scripts
- playable local audio and video practice assets
- the original sample Loom video
- an embedded tab for Claude's HTML version

## Project Structure

- `2-opencode-loom/`
  Main deployable assets and HTML hub.
- `loom-sample/`
  Sample Loom video and supporting analysis artifacts.
- `loom-drafts/`
  Earlier intermediate Loom drafts and alternate HTML outputs.
- `references/`
  Private reference material such as resume, screenshots, guides, and notes.

## Recommended Repo Scope

Commit:

- `2-opencode-loom/`
- `loom-sample/compressed-2.mp4`
- `.gitignore`
- this `README.md`

Optionally commit:

- `loom-sample/compressed-2.transcript.md`
- `loom-drafts/` if you want version history of the drafts

Keep ignored:

- local tool folders and virtualenvs
- private reference materials
- temporary analysis assets
